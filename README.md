# @amlplugins/google-cloud-text-to-speech

Native `@google-cloud/text-to-speech` SDK re-exported under the `@amlplugins` namespace.

> Text-to-Speech.

## Install

```bash
npm install @amlplugins/google-cloud-text-to-speech
```

## Use

```ts
import { /* named exports from @google-cloud/text-to-speech */ } from "@amlplugins/google-cloud-text-to-speech";
// or
import sdk from "@amlplugins/google-cloud-text-to-speech";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
